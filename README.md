# Bardo Manifesto / Essay

Static site for the Bardo manifesto essay. View it on GitHub Pages after following the steps below.

## Publish to GitHub Pages

1. **Create a new repository on GitHub**
   - Go to [github.com/new](https://github.com/new)
   - Name it (e.g. `bardo-manifesto` or `bardo-essay`)
   - Leave it empty (no README, .gitignore, or license)
   - Click **Create repository**

2. **Push this project to it** (run in this folder in Terminal):

   ```bash
   cd "/Users/fabian/Documents/Bardo Projekt Essay - Replica"

   git remote add origin https://github.com/YOUR_USERNAME/YOUR_REPO_NAME.git
   git branch -M main
   git push -u origin main
   ```

   Replace `YOUR_USERNAME` with your GitHub username and `YOUR_REPO_NAME` with the repo name you chose.

3. **Turn on GitHub Pages**
   - On the repo page, go to **Settings** â†’ **Pages**
   - Under **Source**, choose **Deploy from a branch**
   - Branch: **main**, folder: **/ (root)**
   - Click **Save**

4. **Your link**
   - After a minute or two, the site will be at:
   - **https://YOUR_USERNAME.github.io/YOUR_REPO_NAME/**
   - That URL (and the root of the repo) will open the manifesto essay.
